# Module 02 Overview

**Purpose**: The computational core. This is where multivariable calculus becomes a working tool rather than a geometric abstraction.

| Unit | Title | Time |
|---|---|---|
| [01](unit-01-partial-derivatives) | First-Order Partial Derivatives | 55–65 min |
| [02](unit-02-higher-order) | Higher-Order Partials & Clairaut's Theorem | 50–60 min |
| [03](unit-03-total-derivative) | The Total Derivative & Tangent Planes | 55–65 min |
| [04](unit-04-chain-rule) | The Multivariable Chain Rule | 55–65 min |

## Module Learning Objectives

- Compute first- and higher-order partial derivatives of polynomial, exponential, and logarithmic functions
- State Clairaut's theorem and recognise when mixed partials commute
- Construct tangent plane approximations and interpret the total differential
- Apply the multivariable chain rule to compute gradients of composed functions
