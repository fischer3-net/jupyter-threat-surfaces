# Overview

**Purpose**: Assemble partial derivatives into a vector, establish its geometry, and connect the gradient to optimization and statistical estimation.

| Unit | Title | Time |
|---|---|---|
| [01](unit-01-gradient-vector) | The Gradient Vector | 55–65 min |
| [02](unit-02-directional-derivatives) | Directional Derivatives & Steepest Ascent | 50–60 min |
| [03](unit-03-jacobian) | The Jacobian | 55–65 min |

## Module Learning Objectives

- Define the gradient vector and compute it for functions of two and three variables
- Interpret the gradient geometrically as the direction of steepest ascent and as normal to level sets
- Compute directional derivatives and identify the direction that maximizes or minimizes the rate of change
- Construct the Jacobian matrix for a vector-valued function and apply the multivariable chain rule
- Recognize the gradient as the central object in both gradient-based optimization and maximum likelihood estimation

## Prerequisites

This module builds directly on **Module 02 — Partial Derivatives & Differentiability**. You must be fluent with computing partial derivatives before proceeding. If partial derivatives feel uncertain, return to Module 02 before continuing.
